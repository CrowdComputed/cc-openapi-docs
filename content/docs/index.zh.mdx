---

### 第 1 步：注册您的账户(免费体验 1700 积分)

首先，您需要一个 CrowdComputed 帐户。注册是免费的，注册成功后您将立即获得平台的访问权限和 1700 积分，用以体验我们的 API。

- **点击此处进行注册:** [**https://crowdcomputed.com/zh/login/**](https://crowdcomputed.com/zh/login/)

---

### 第 2 步：获取您的 API 密钥

注册并登录后，请点击官网右上角，弹出菜单的 **“API”** 选项。在弹出的模态框中，您将看到您专属的 API 密钥。该密钥将用于验证您的所有 API 请求。
![image.png](https://api.apifox.com/api/v1/projects/7019373/resources/568202/image-preview)

> **重要提示:** 请像对待密码一样妥善保管您的 API 密钥，确保其安全，切勿在公开场合分享。

---

### 第 3 步：进行您的第一次 API 调用

现在您可以开始集成了！使用上一步获取的 API 密钥，您就可以向我们的 API 接口发起认证请求。请查阅我们完整的 API 参考文档，以获取关于可用接口、必需参数以及生成图像、视频等功能的详细信息和具体示例。

[Generation Workflows](apifox://link/folders/65194300)

---

### 第 4 步：升级订阅以解锁更多积分/功能

您的免费体验积分非常适合用于测试和初期开发。当您准备将服务用于生产环境时，可以随时升级到我们高性价比的订阅计划，以获取更多的生成积分和高级功能。

- **访问我们的订阅页面查看并升级您的计划:** [**https://crowdcomputed.com/zh/subscription/**](https://crowdcomputed.com/zh/subscription/)

---

### 如何在文档页面上测试我们的 API

在每个接口文档页面，可以直接点击“Try it”按钮，并输入上一步获取的 API 密钥，您就可以向我们的 API 接口发起请求。您只需要输入一次 token，其他接口文档页面就可以记录您的输入历史并自动填充 token 字段。

![try-it.png](https://api.apifox.com/api/v1/projects/7019373/resources/600913/image-preview)

![input-token.png](https://api.apifox.com/api/v1/projects/7019373/resources/600914/image-preview)

如果您已经通过点击“Run in Apifox”按钮进入 Apifox 页面，可以点击右上角的"管理环境变量"按钮。

![set-env.png](https://api.apifox.com/api/v1/projects/7019373/resources/600920/image-preview)

在“正式环境”中配置您的全局 token 变量。

![set-api-key.png](https://api.apifox.com/api/v1/projects/7019373/resources/600921/image-preview)

这样您后续发起任何接口调用时，只要在 Auth 选项卡中选择了“使用接口默认鉴权凭证”，就可以使用您配置的全局 token 来发起 API 请求。

![request-auth.png](https://api.apifox.com/api/v1/projects/7019373/resources/600925/image-preview)
